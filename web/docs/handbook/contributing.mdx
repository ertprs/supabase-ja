## 貢献するには

Supabase has many [open source repos](https://github.com/orgs/supabase/repositories). Some of the more popular ones are:
  - [Supabase](https://github.com/supabase/supabase) - Our Dashboard, Websites, and example apps
  - [Supabase-js](https://github.com/supabase/supabase-js) - A clientside library written in Typescript
  - [Gotrue](https://github.com/supabase/gotrue) - Our認証server, written in Go
  - [Realtime](https://github.com/supabase/realtime) - A server for streaming changes from Postgres, written in Elixir 
  - [browse all](https://github.com/orgs/supabase/repositories)

貢献してみませんか。私たちのGitHubリポジトリーに参加しましょう。

- [Supabaseをスポンサーする](https://github.com/sponsors/supabase)。
- [Disucussionsに答える](https://github.com/supabase/supabase/discussions)。
- いずれかの[リポジトリー](https://github.com/orgs/supabase/repositories)にIssueを送る。
- パフォーマンスに関する問題や、ドキュメントの分かりにくい部分を報告してください。
- いずれかの[レポジトリー](https://github.com/orgs/supabase/repositories)にプルリクエストを作成する。
- [Readmeを翻訳する](https://github.com/supabase/supabase/issues/1341)。
- プロダクトを使ってみて、フィードバックしてください。
- 私たちの活動を気に入っていただけましたら、評判を広めてください。
