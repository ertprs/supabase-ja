## 貢献するには

貢献してみませんか。私たちのGitHubリポジトリーに参加しましょう。

- [Supabaseをスポンサーする](https://github.com/sponsors/supabase)。
- [Disucussionsに答える](https://github.com/supabase/supabase/discussions)。
- Issueを投稿する。
- パフォーマンスに関する問題や、ドキュメントの分かりにくい部分を報告してください。
- プルリクエストを作成する。
- [Readmeを翻訳する](https://github.com/supabase/supabase/issues/1341)。
- プロダクトを使ってみて、フィードバックしてください。
- 私たちの活動を気に入っていただけましたら、評判を広めてください。
