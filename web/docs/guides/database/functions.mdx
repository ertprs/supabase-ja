## 関数の作成

Supabaseは、データベースの関数を作成するためのいくつかのオプションを提供しています。
ダッシュボードを使用するか、SQLを使用して直接作成できます。ダッシュボードにはSQLエディターが用意されていますが、データベースに[接続し](/docs/guides/database/connecting/connecting-to-postgres)て自分でSQLクエリを実行できます。


1. SQLエディターセクションに移動します。
2. 「New Query」をクリックします。
3. データベース関数を作成または置換するためのSQLを入力します。
4. 「Run」をクリックするか、cmd+enter (ctrl+enter)をクリックします。

### 基本関数

「hello world」という文字列を返す基本的な例です。

```sql
create or replace function hello_world() -- 1
returns text -- 2
language sql -- 3
as $$  -- 4
  select 'hello world';  -- 5
$$; --6

```

<details>
<summary>詳細の表示/非表示</summary>

最も基本的な関数は、以下の部分で構成されています。

1. `関数 hello_world() を作成または置換し`ます。`hello_world`は関数の名前です。新規に関数を作成する場合は`create`、既存の関数を置き換える場合は`replace`を使用します。また、`createとreplace`を併用することで、どちらかを処理できます。
2. は`テキストを返し`ます。この関数が返すデータの種類です。何も返さない場合は、`voidを返し`ます。
3. `language sql`:関数本体で使用される言語です。`plpgsql`、`plv8`、`plpython`などの手続き型言語の場合もあります。
4. `as $$:`関数のラッパーです。`$$`記号で囲まれたものは、関数本体の一部となります。
5. `select 'hello world'; :` 単純な関数本体です。関数本体内の最後の`select`文は、それに続く文がない場合に返されます。
6. `$$; :` 関数ラッパーの閉じたシンボルです。

</details>

### テーブルセットを返す

例題のデータセットに含まれるすべての`惑星`を返す基本的な例です。

```sql
create or replace function get_planets() 
returns setof planets
language sql
as $$
  select * from planets;
$$;
```

#### 実行

この関数はテーブルセットを返すので、フィルタやセレクターを適用できます。例えば、最初の惑星だけが欲しい場合は次のようにします。

### パラメーター付き

新しい惑星を`planets`に挿入し、新しいIDを返す基本的な例です。今回は`plpgsql`という言語を使っていることに注意してください。

```sql
create or replace function add_planet(name text) 
returns bigint
language plpgsql
as $$
declare 
  new_row bigint;
begin
  insert into planets(name)
  values (add_planet.name)
  returning id into new_row;

  return new_row;
end;
$$;
```

#### 実行時

この関数はテーブルセットを返すので、フィルタやセレクターを適用できます。例えば、最初の惑星の`名前`だけが欲しい場合は次のようにします。
