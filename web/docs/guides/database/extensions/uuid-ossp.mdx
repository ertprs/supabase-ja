`UUID`は「Universally Unique Identifer」の略です。


## 概要

`UUID`とは "Universally Unique Identifer "の略で、実用的に一意であるため、主キーとして特に適しています。また、`GUID`と呼ばれることもありますが、これは「Globally Unique Identifer」の略です。

### `uuid`タイプ

この拡張機能を有効にすると、`uuid`型へアクセスできるようになります。

### `uuid_generate_v1()`

コンピューターのMACアドレス、現在のタイムスタンプ、そしてランダムな値の組み合わせに基づいてUUID値を作成します。

### `UUID_GENERATE_V4()`

乱数のみに基づいてUUID値を作成します。

## 例

### クエリの中で

```sql
select uuid_generate_v4();
```

### 主キーとして

テーブルに一意のランダムなIDを自動的に作成します。

```sql
create table contacts (
  id uuid default uuid_generate_v4(),
  first_name text,
  last_name text,

  primary key (id)
);
```
