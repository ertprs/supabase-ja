あなたのプロジェクトでSpotify Authを有効にするには、Spotify OAuthアプリケーションを設定します。そして、アプリケーションの認証情報をSupabaseダッシュボードに追加する必要があります。

## 概要

あなたのアプリケーションにSpotifyのログインを設定するには、3つのパートで構成されています。

- [Spotify開発者ダッシュボード](https://developer.spotify.com/dashboard/)で、Spotifyプロジェクトとアプリを作成・設定する。
- [Supabaseプロジェクト](https://supabase.com)に、Spotify`API Key`と`API Secret Key`を追加する。
- [Supabase JS Client App](https://github.com/supabase/supabase-js)にログインコードを追加します。

## 手順

### あなたのSpotify Developerアカウントにアクセスします

- [Spotify](https://spotify.com)にログインする。
- Spotify[開発者ダッシュボード](https://developer.spotify.com/dashboard)にアクセスします。

![Spotify Developer Portal.](/img/guides/spotify-portal.png)

### コールバックURLを探す

次のステップでは、次のようなコールバックURLが必要です。

`https://<project-ref>.supabase.co/auth/v1/callback`

- [Supabaseのプロジェクトダッシュボード](https://supabase.com)にアクセスします。
- 左サイドバーの下部にある`「設定」`アイコンをクリックします。
- リストの中の「`API`」をクリックします。
- Config / URLの下にあなたのAPI URLがありますので、`Copy`をクリックしてクリップボードにコピーします。
- 最後に`/auth/v1/callback`を追加すれば、完全な`OAuthリダイレクトURI`が得られます。

### Spotify OAuthアプリを作成する

- [Spotify](https://spotify.com)にログインします。
- Spot[ify Developer Dashboard](https://developer.spotify.com/dashboard)にアクセスします。
- `Create an App`をクリックします。
- `アプリの名前`を入力してください。
- `アプリの説明`を入力してください。
- `デベロッパーTOSとブランディングガイドライン`に同意する場合は、チェックを入れてください。
- `作成`をクリックします。
- `クライアントID`を保存する。
- `クライアントシークレット`の保存します。
- 「`Edit Settings`」をクリックします。

`リダイレクト URI`をクリックします。

- SupabaseのコールバックURLを入力してください。
- `追加`をクリックします。
- 下部の「`Save`」をクリックします。

### SupabaseプロジェクトにSpotifyの認証情報を入力します

- [Supabaseのプロジェクトダッシュボード](https://supabase.com)にアクセスします。
- 左側のサイドバーで`「認証」`アイコンをクリックします（上部付近）。
- リストから`「設定」`をクリックして、`「認証設定」`ページを表示します。
- 「`Site URL`」にアプリの最終的な（ホストされた）URLを入力します（これは重要です）。
- 「`External OAuth Providers`」の項目で、「`Spotify Enabled`」をオンにします。
- 前のステップで保存した`API Key``(client_id`)と`API Secret Key``(client_secret`)を入力します。
- 「`Save`」をクリックします。

### クライアントアプリにログインコードを追加する

JavaScriptのクライアントコードはこちらに記載されています。[Supabase OAuth Client Code](/docs/reference/javascript/auth-signin#sign-in-using-third-party-providers).

```js
const { user, session, error } = await supabase.auth.signIn({
  provider: 'spotify',
})
```

ボタン、リンク、またはUI要素から呼びだすことができるこの関数を追加します。

```js
async function signInWithSpotify() {
  const { user, session, error } = await supabase.auth.signIn({
    provider: 'spotify',
  })
}
```

ログアウトするには次の関数を追加します。

```js
async function signout() {
  const { error } = await supabase.auth.signOut()
}
```
